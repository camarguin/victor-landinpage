# Victor Gerhardt Landing Page

Landing page estática em HTML e CSS para advocacia criminal.

## Estrutura

- `index.html`: marcação completa da página.
- `styles.css`: estilos responsivos, animações e componentes visuais.
- `public/`: favicon, imagem principal e demais assets.

## Como visualizar

Abra o arquivo `index.html` diretamente no navegador.

Também é possível hospedar a pasta em qualquer serviço de arquivos estáticos, como Netlify, Vercel, GitHub Pages ou um servidor HTTP simples.
